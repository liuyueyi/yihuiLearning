## 2. img模板

模板格式说明，json格式；主要分四块资源配置

- draw: 二维码0/1点对应的资源
- logo: logo对应的配置信息
- bg: 背景图资源参数
- ft: 前置图资源参数

```json
{
  "draw": [
      {
        "type": "pre",
        "size": "1x1",
        "miss": "",
        "img": ""
      }
  ]

}

```
