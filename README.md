## Intro-介绍

由于个人主站博文数量700+，并有持续增多的趋势，原有基于hexoblog搭建的博文，实在无法将博文信息有效管理起来，一个是编译生成静态网站慢，另一个原因则是碎片化太严重，没有将系列博文整合在一起不便于学习使用

因此决定使用Hugo + Doscify来重新搭建一个站点，同时也花了大量的精力，将之前的博文进行分类管理，整理处一系列的教程，方便有需要的小伙伴阅读
